# MU-MIMO plots

The matrix marks stations that are configured for one spatial stream and
actually receive application traffic in each condition. The companion panel
shows measured aggregate goodput. A wider or higher-dimensional MU-MIMO setup
should serve more users concurrently and increase aggregate delivery.

Per-PPDU spatial-stream assignments are not currently exported as result
vectors. Consequently, the matrix is a configuration-and-delivery
compatibility view, not observed beamforming telemetry. The title states this
distinction explicitly.
